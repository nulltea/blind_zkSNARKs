.. _library:

Library
=======

To use the library from an application, you need to include the header file:

.. code-block:: C
    
    #include "lazer.h"

When linking, you need to link either `liblazer.a` or `liblazer.so`.

This section documents the library's interfaces and the script that generates parameter set header files. 
Also examples are provides and a complete reference for the library.
